# REPORTING ISSUES

If you wish to report an issue with this library, please use
[this link](https://github.com/MrYsLab/telemetrix-rpi-pico-2w/issues).

For servers, please use
[this link](https://github.com/MrYsLab/Telemetrix4RpiPico2w/issues).

When filing an issue, please provide:

1. Any console output.
2. A description of the issue.
3. Sample code demonstrating the problem.
